SECTION B — 2 Marks Each

Q6.   Rational Numbers between 1 and 2   [2 Marks]

→ Let us write 1 and 2 with denominator \(4\): \(1 = \frac{4}{4}\) and \(2 = \frac{8}{4}\).

→ Three rational numbers between them are \(\frac{5}{4}, \frac{6}{4}, \frac{7}{4}\).

Marking: 1 mark for identifying equivalent fractions with a larger denominator. 1 mark for providing the numbers. Accept: Any three correct mathematical values (e.g., 1.1, 1.2, 1.5).

Ref: NCERT Class 9 Maths, Ch. 1, Ex. 1.1

Q7.   \(p/q\) conversion   [2 Marks]

→ Let \(x = 0.333...\) (eq 1)

→ Multiply by 10: \(10x = 3.333...\) (eq 2)

→ Subtract eq 1 from eq 2: \(9x = 3\)

→ \(x = \frac{3}{9} = \frac{1}{3}\)

Marking: 1 mark for setting up the 10x equation. 1 mark for correct subtraction and simplification to \(\frac{1}{3}\).

Ref: NCERT Class 9 Maths, Ch. 1, Ex. 1.3

SECTION C — 3 Marks Each

Q8.   Rationalisation   [3 Marks]

→ Multiply numerator and denominator by the conjugate \(\sqrt{5} - \sqrt{2}\).

→ \(\frac{1}{\sqrt{5} + \sqrt{2}} \times \frac{\sqrt{5} - \sqrt{2}}{\sqrt{5} - \sqrt{2}}\)

→ \( = \frac{\sqrt{5} - \sqrt{2}}{(\sqrt{5})^2 - (\sqrt{2})^2} \)

→ \( = \frac{\sqrt{5} - \sqrt{2}}{5 - 2} = \frac{\sqrt{5} - \sqrt{2}}{3} \)

Marking: 1 mark for selecting the correct conjugate. 1 mark for correct denominator expansion (\(5-2\)). 1 mark for final expression.

Ref: NCERT Class 9 Maths, Ch. 1, Ex. 1.5

Q9.   Locating \(\sqrt{13}\) on number line   [3 Marks]

→ By Pythagoras theorem, \(\sqrt{13} = \sqrt{3^2 + 2^2}\).

→ Draw a horizontal number line and mark point A at 0 and point B at 3.

→ Construct a perpendicular BC of 2 units at point B.

→ Join AC. The length of AC is \(\sqrt{3^2 + 2^2} = \sqrt{9+4} = \sqrt{13}\) units.

→ With A as center and AC as radius, draw an arc to intersect the number line at P. Point P represents \(\sqrt{13}\).

Marking: 1 mark for identifying components (3 and 2). 1 mark for drawing the right-angled triangle. 1 mark for dropping the arc.

Ref: NCERT Class 9 Maths, Ch. 1, Ex. 1.2

SECTION D — 5 Marks

Q10.   Application - Rationalisation   [5 Marks]

→ (a) Expression is \(\frac{4 + \sqrt{5}}{4 - \sqrt{5}}\). Rationalise by multiplying numerator and denominator by \((4 + \sqrt{5})\).

→ \(\frac{(4 + \sqrt{5})(4 + \sqrt{5})}{(4 - \sqrt{5})(4 + \sqrt{5})} = \frac{(4 + \sqrt{5})^2}{16 - 5}\)

→ \( = \frac{16 + 5 + 8\sqrt{5}}{11} = \frac{21 + 8\sqrt{5}}{11} \)

→ (b) Therefore, comparing with \(a + b\sqrt{5}\): \(a = \frac{21}{11}\) and \(b = \frac{8}{11}\).

→ (c) Calculate approximate value: \( \frac{21 + 8(2.236)}{11} = \frac{21 + 17.888}{11} \)

→ \( = \frac{38.888}{11} \approx 3.535 \)

Marking: 1 mark for multiplying by conjugate. 1 mark for numerator expansion (\((A+B)^2\)). 1 mark for arriving at final fraction. 1 mark for correctly extracting a and b. 1 mark for decimal approximation.

Ref: NCERT Class 9 Maths, Ch. 1, Ex. 1.5
