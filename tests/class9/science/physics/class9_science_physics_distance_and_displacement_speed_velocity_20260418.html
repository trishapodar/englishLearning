Yes. If a body moves with uniform speed in a circular path, its direction continuously changes, so it is accelerated.

SECTION B — 2 Mark(s) Each

Q6.   Distance: 7m, Disp: 5m   [2 Mark(s)]

Distance = 3 m + 4 m = 7 m.

→ Displacement = √(3² + 4²) = √(9+16) = √25 = 5 m.

Marking: 1 mark for distance (7m). 1 mark for displacement with Pythagoras (5m).
Accept: Valid vector diagram solving to 5m.
Common Error: Adding segments outright for displacement without recognizing the right-angle triangle.

Ref: NCERT Class 9 Science, Ch. 8

Q7.   1/54 m/s² or 1.85 cm/s²   [2 Mark(s)]

u = 0, v = 40 km/h = 40 × 5/18 = 100/9 m/s.

→ t = 10 min = 600 s.

→ a = (v-u)/t = (100/9) / 600 = 1/54 m/s² ≈ 0.0185 m/s².

Marking: 1 mark for converting to valid SI units. 1 mark for correct formula application and answer.
Accept: Answer left in km/h² if explicitly specified, e.g. 240 km/h².
Common Error: Failure to convert 10 minutes to hours or seconds before plugging into formula.

Ref: NCERT Class 9 Science, Ch. 8, In-text Q

SECTION C — 3 Mark(s) Each

Q8.   Average speed is 66.67 km/h   [3 Mark(s)]

Total distance = 100 + 100 = 200 km.

→ Time 1 = 100/50 = 2 hr. Time 2 = 100/100 = 1 hr. Total time = 3 hr.

→ Average Speed = 200/3 = 66.67 km/h.

→ Simply averaging speeds (50+100)/2 = 75 km/h is incorrect because the car spends unequal amounts of time travelling at those speeds.

Marking: 1 mark for total duration (3 hr). 1 mark for correct average speed formula and answer. 1 mark for the conceptual reasoning.
Accept: 200/3 km/h fraction form.
Common Error: Taking the arithmetic mean of 50 and 100 resulting in 75 km/h.

Ref: NCERT Class 9 Science, Ch. 8

Q9.   Max Height = 80m, Disp=0, Dist=160m   [3 Mark(s)]

u = 40 m/s, v = 0 (at highest point), a = -10 m/s².

→ Using v² - u² = 2as, 0 - 1600 = -20s => s = 80 m. Max height = 80 m.

→ Displacement upon return = 0 m.

→ Total distance = 80 m (up) + 80 m (down) = 160 m.

Marking: 1 mark for Max Height. 1 mark for Displacement = 0. 1 mark for Total Distance = 160m.
Accept: Accept logical derivations using time of flight.
Common Error: Forgetting that acceleration due to gravity is negative during upward flight; stating distance equals displacement.

Ref: NCERT Class 9 Science, Ch. 8

SECTION D — 5 Mark(s) Each

Q10.   v=20m/s, a=-4m/s², dist=750m   [5 Mark(s)]

(a) Acceleration phase: u = 0, a = 2, t = 10. v = u+at = 20 m/s. Max velocity is 20 m/s.

(b) Braking phase: u = 20, v = 0, t = 5. a = (0-20)/5 = -4 m/s². Retardation is 4 m/s².

(c) V-T Graph details: (0,0) to (10, 20) straight line up. (10, 20) to (40, 20) flat line. (40, 20) to (45, 0) straight line down.

(d) Total distance = Area under the v-t graph (a trapezium).
=> Area = 1/2 × (Sum of parallel sides) × height = 1/2 × (30 + 45) × 20 = 75 × 10 = 750 m.

Marking: (a) 1 mark. (b) 1 mark for correct finding of retardation. (c) 1 mark for correct shape showing acceleration, steady state, deceleration. (d) 2 marks for obtaining 750m using area or kinematic formulas.
Accept: Computing part d area via 3 distinct shapes (triangle + rectangle + triangle) = 100 + 600 + 50 = 750 m.
Common Error: Miscalcuating the flat cruising time interval (using 30 instead of 40 at the end).

Ref: NCERT Class 9 Science, Ch. 8
