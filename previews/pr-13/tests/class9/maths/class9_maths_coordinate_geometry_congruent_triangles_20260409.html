Q3.   Answer: SSS (Side-Side-Side) Congruence Criterion   [1 mark]

If three sides of one triangle are equal to the three corresponding sides of another triangle, the triangles are congruent by the SSS criterion.

Marking: Writing "SSS" alone — 1 mark. Writing an incorrect criterion — 0 marks.
Q4.   Answer: (B) ∠E   [1 mark]

If △ABC ≅ △DEF, then A↔D, B↔E, C↔F. Therefore ∠B corresponds to ∠E.

Marking: Correct option — 1 mark.

Q7 — Isosceles triangle angles   [2 marks]
Property used: Angles opposite to equal sides of an isosceles triangle are equal (Base angles theorem). [½ mark]
Since AB = AC: ∠ABC = ∠ACB [½ mark]
Angle sum: ∠BAC + ∠ABC + ∠ACB = 180°  ⟹  50° + 2∠ABC = 180° [½ mark]
∴ ∠ABC = ∠ACB = 65° [½ mark]
Marking: Property stated — ½ mark | Setting up equation — ½ mark | Each correct angle — ½ mark each.
Q8 — Congruence criterion   [2 marks]
△PQR ≅ △LMN by SSS congruence (all three sides equal). [1 mark]
Corresponding parts: P↔L, Q↔M, R↔N; PQ=LM, QR=MN, PR=LN; ∠P=∠L, ∠Q=∠M, ∠R=∠N. [1 mark]
Marking: Correct criterion named — 1 mark. All six CPCT pairs correctly listed — 1 mark (accept any 3 pairs for full credit here).

Q13 — Prove △ABC isosceles (D midpoint, AD⊥BC)   [3 marks]
In △ADB and △ADC:
DB = DC   (D is midpoint of BC)   [½ mark]
AD = AD   (Common side)   [½ mark]
∠ADB = ∠ADC = 90°   (AD ⊥ BC)   [½ mark]
∴ △ADB ≅ △ADC   by SAS congruence   [½ mark]
∴ AB = AC   (CPCT)   ∴ △ABC is isosceles.   [1 mark]
Marking: Each pair of corresponding equal parts — ½ mark each | SAS named — ½ mark | AB = AC by CPCT — 1 mark.
Q14 — ∠BCD = 90°   [3 marks]
Given: AB = AC (isosceles), AD = AB. ∴ AC = AD. [½ mark]
In △ABC: AB = AC ⟹ ∠ABC = ∠ACB. Let ∠ABC = ∠ACB = x. [½ mark]
In △ACD: AC = AD ⟹ ∠ADC = ∠ACD = y. [½ mark]
In △BCD: ∠DBC + ∠BCD + ∠BDC = 180°. [½ mark]
∠BCD = ∠ACB + ∠ACD = x + y.   ∠DBC = x, ∠BDC = y. [½ mark]
x + (x + y) + y = 180° ⟹ 2x + 2y = 180° ⟹ x + y = 90°. ∴ ∠BCD = 90°. [½ mark]
Marking: Each correct logical step — ½ mark (6 steps × ½ = 3 marks). Award marks for correct reasoning even if final line is not reached.

Q19 — ∠BOC = 90° + ½∠A   [5 marks]
Let ∠A = 2α, ∠B = 2β, ∠C = 2γ (since OB and OC are angle bisectors). [1 mark — setting up notation]
In △ABC: ∠A + ∠B + ∠C = 180° ⟹ 2α + 2β + 2γ = 180° ⟹ α + β + γ = 90°. [1 mark]
∠OBC = ½∠B = β   and   ∠OCB = ½∠C = γ. [1 mark]
In △BOC: ∠BOC + ∠OBC + ∠OCB = 180°. [½ mark]
∠BOC = 180° − β − γ. [½ mark]
Since α + β + γ = 90°, we have β + γ = 90° − α. [½ mark]
∴ ∠BOC = 180° − (90° − α) = 90° + α = 90° + ½∠A.   ∎ [½ mark]
Marking: Setting up bisector angles — 1 mark | Using angle sum in △ABC — 1 mark | Expressing ∠OBC, ∠OCB — 1 mark | Angle sum in △BOC — ½ mark | Substitution and final result — 1½ marks. Award marks for method even if subscript notation differs.
Q20 — △PQR with PS⊥QR, S midpoint   [5 marks]
(i) In △PQS and △PRS:
QS = RS (S midpoint) | PS = PS (common) | ∠PSQ = ∠PSR = 90° (PS⊥QR) [1 mark]
∴ △PQS ≅ △PRS by SAS (two sides and included right angle). [½ mark]
(ii) PQ = PR (CPCT) ∴ △PQR is isosceles. [½ mark]
(iii) ∠QPR = ∠QPS + ∠RPS. By CPCT, ∠QPS = ∠RPS. ∴ ∠QPR = 2∠QPS. [1 mark]
(iv) ∠Q = ∠R = 55° (base angles, isosceles). ∠QPR = 180° − 55° − 55° = 70°. [1 mark]
(v) △PQS ≅ △PRS. Corresponding parts: PQ=PR, QS=RS, PS=PS, ∠Q=∠R, ∠QPS=∠RPS, ∠PSQ=∠PSR. [½ mark]
Marking: Three pairs for congruence — 1 mark | SAS + congruence statement — ½ | PQ=PR — ½ | Proof ∠QPR = 2∠QPS — 1 mark | All angles in (iv) — 1 mark | CPCT list — ½ mark.
