SECTION A — 1 Mark Each

Q1.   B   [1 Mark]

The alphabet B has exactly one horizontal line of symmetry.

Marking: 1 mark for correct alphabet. Common Error: Choosing X (which has both horizontal and vertical).
Q2.   30, 34   [1 Mark]

The rule is +4. So 26 + 4 = 30, and 30 + 4 = 34.

Marking: 0.5 mark each for 30 and 34.
Q3.   A Circle   [1 Mark]

When viewing a cylinder from the top, only the circular lid/base is visible.

Marking: 1 mark for identifying the circle.
Q4.   Tiling   [1 Mark]

Tessellation means covering a surface with flat shapes so that there are no gaps or overlaps.

Marking: 1 mark for explaining "no gaps or overlaps".
Q5.   MOM   [1 Mark]

The mirror image of "MOM" remains "MOM" due to its vertical symmetry.

Marking: 1 mark for correct image.

SECTION B — 2 Marks Each

Q6.   Lunch Box Views   [2 Marks]

→ Front view: A narrow rectangle representing the side height and full length.

→ Top view: A larger flat rectangle representing the lid.

Marking: 1 mark for correct front view sketch, 1 mark for top view sketch.
Q7.   Hexagon, Heptagon, Octagon   [2 Marks]

→ Rule: Each shape gains one more side (3 sides, 4 sides, 5 sides...).

→ Next shapes: 6-sided (Hexagon), 7-sided (Heptagon), 8-sided (Octagon).

Marking: 1 mark for stating the correct rule. 1 mark for the three correct shapes.

SECTION C — 3 Marks Each

Q8.   Classroom Symmetry   [3 Marks]

→ Example 1: A chalkboard (Rectangle, 2 lines of symmetry).

→ Example 2: A square window (Square, 4 lines of symmetry).

→ Example 3: An analog clock (Circle or regular polygon, multiple lines).

Marking: 1 mark per valid object with correctly drawn lines of symmetry.
Q9.   Number Tower   [3 Marks]

→ Bottom row: 2, 4, 3, 5

→ Row 3: (2+4)=6, (4+3)=7, (3+5)=8

→ Row 2: (6+7)=13, (7+8)=15

→ Top (Row 1): (13+15)=28

Marking: 1 mark for correctly calculating Row 3. 1 mark for Row 2. 1 mark for Top row.

SECTION D — 5 Marks
